ШИФРАТОР \ ДЕШИФРАТОР СООБЩЕНИЙ ШИФРА ЦЕЗАРЯ (Требуется Java 25+)

Как он работает: 

1) При запуске программы вам выйдет приветствующее сообщение и запросит от вас действие, зашифровать или расшифровать сообщение (при расшифровке потребует ключ).

2) Оператору необходимо ввести "Encode" - зашифровать или "Decrypt" - расшифровать.
    
3.1) Если оператор выбрал "Encode", то затем требуется ввести фразу, которую нужно зашифровать, через какое-то время код полностью зашифрует сообщение и выведет его вам на экран с ключом для расшифровки.

    
3.2) Если оператор выбрал "Decrypt", то программа затребует сообщение, а затем ключ который сгенерирован по "принципу программы" или который она вам раннее дала. Затем пойдет процесс расшифровки и вам выдаст сообщение в расшифрованном виде.

3) Сделать более продуманную генерацию ключа к сообщению, так как ещё имеется возможность вычленить из него "trueKey" вручную.
4) Убрать "Debug" Функционал в Main.java, он был мне нужен для отслеживания ошибок, поэтому выдает лишние сообщения, по типу trueKey.
